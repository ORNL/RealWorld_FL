# RealWorld_FL

## TPC26 Birds-of-a-Feather

**Trustworthy Privacy-Preserved Federated Learning for Science** — Trillion Parameter Consortium 2026,
Baltimore, Maryland, June 2, 2026.

A page recording the session, its nine lightning talks, and how to get involved lives in
[`docs/index.html`](docs/index.html). It is a single self-contained file with no build step and no
external assets — open it directly, or serve `docs/` with GitHub Pages.

Organized by Olivera Kotevska (ORNL), Kibaek Kim (ANL), and Ravi Madduri (ANL).
To join the effort, contact <kotevskao@ornl.gov>.
